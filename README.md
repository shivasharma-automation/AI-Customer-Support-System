# AI Customer Support System

An AI-powered Customer Support Automation System built with **n8n**, **Groq AI**, **Gmail**, **Google Sheets**, **HTML**, **CSS**, and **JavaScript**.

---

# Features

- Customer Support Form
- AI Ticket Analysis
- Category Detection
- Priority Detection
- AI-Generated Customer Reply
- Automatic Ticket ID Generation
- Google Sheets Ticket Logging
- Human Escalation
- Gmail Notifications
- End-to-End Automation using n8n

---

# Tech Stack

- n8n v2.27.4
- Groq AI (Llama 3.1 8B Instant)
- Gmail
- Google Sheets
- HTML
- CSS
- JavaScript

# Installation

1. Clone the repository.
2. Import the workflow into n8n.
3. Configure:
   - Groq API Key
   - Gmail Credentials
   - Google Sheets Credentials
4. Open `frontend/index.html`.
5. Submit a support request.
